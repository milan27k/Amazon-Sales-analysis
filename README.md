# Amazon-Sales-analysis
# 📊 Amazon Sales Analysis Dashboard

## 🔷 Project Overview

This project analyzes Amazon sales data to uncover key business insights related to revenue, customer demand, and operational performance.
The analysis is performed using **Python (data cleaning), SQL (analysis), and Power BI (dashboard visualization)**

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing categories and locations
* Evaluate cancellation impact and revenue loss
* Compare fulfilment efficiency (Amazon vs Merchant)
* Understand customer purchasing behavior

## 🛠️ Tools & Technologies

* 🐍 Python (Pandas, NumPy) – Data Cleaning
* 🗄️ SQL – Data Analysis
* 📊 Power BI – Dashboard Visualization

## 🧹 Data Cleaning (Python)

* Removed duplicate records
* Handled missing values using context-based imputation
* Replaced placeholder values (e.g., ‘…’) with standardized values
* Standardized categorical columns (city, fulfilment, etc.)

## 🗄️ SQL Analysis

Key business questions solved using SQL:

* Total sales and order count
* Average order value
* Top categories and cities by revenue
* Cancellation rate and revenue loss
* Fulfilment performance comparison
* Order value segmentation

## 📊 Dashboard (Power BI)

### 🔹 Page 1: Sales Overview

* Key metrics (Sales, Orders, AOV, Cancellation Rate)
* Category and regional performance
* Fulfilment distribution
* Revenue loss insights

### 🔹 Page 2: Sales Performance

* Revenue vs Quantity analysis
* Size and category demand patterns
* City-level performance
* Cancellation impact

### 🔹 Page 3: Operations Analysis

* Cancellation hotspots
* High risk region
* Revenue loss by category
* Order value distribution

## 🔍 Key Insights

* T-Shirts and shirts drives the majority of revenue
* Sales are concentrated in top cities and states
* Cancellation impacts revenue significantly
* Amazon fulfilment shows better delivery performance
* Most orders fall in mid-value range (₹500–₹1000)

## 🚀 Conclusion

This project demonstrates end-to-end data analysis, from cleaning raw data to generating actionable business insights through SQL and interactive dashboards.
